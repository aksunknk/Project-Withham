# withham v1 UI アーカイブ

再利用する React（Vite）＋ Tailwind の UI 資産は **`withham_front/withham-frontend/`** に集約しています。

```bash
cd withham_front/withham-frontend
npm install
npm run dev
```

`.venv` が残っている場合はローカルで削除して構いません（Python バックエンドは本アーカイブから除去済みです）。
